03 · The accessibility baseline

SODA-032, measured

  • Muted token. --t2 is #8A8A8A in the foundation (confirmed live in the rendered kit), in the guideline card, and in the legacy profile simulation. 5.47 to 1 on canvas, 5.04 to 1 on cards. AA clears.
  • Touch targets, measured in the DOM: chips 44px, survey scale cells 44px, stars 45px, the settings gear 44px, the follow-ups dismiss 44px, room view toggles 44px. The DS Chip component carries the same floor for consuming projects.
  • Focus. A global :focus-visible ring in the foundation: 2px green outline, 2px offset, on every interactive element in every kit.
  • Keyboard. Person cards, room list rows, and contact rows are real buttons now (confirmed). The photo step is a labelled button, the hidden code input has a label, the comment sheet and the DS BottomSheet declare dialog semantics, the tab bar marks the active tab, survey stars and scale carry group labels.
  • Decision-log corrections found on the way. SODA-007 reserves purple for the nudge and amber for timers. The Send-Off waitlist button was purple and is now the standard green primary. The Home's cooling-contact dots were amber and are now neutral. The amber token comment now says timers only.
04 · The truth-up

The tracker matches the kits

  • Six statuses restated in the component inventory, in all three of its forms (markdown, desktop render, mobile render): the Room Panel, Survey Panel, Prompt and Pairing Settings, Question Creator, Open Mic Feed, and Broadcast and Announce now read "Prototyped. Rendered in the admin simulation only, not yet in the production kit." The inventory's status note now states the SODA-032 vocabulary: Built means the kit, Prototyped means a simulation, Specified means paper.
  • The orphans are gone. The divergent edit in the root files forced every Admin button to exactly 170px. It is reconciled into the kit as a minimum width of 170px on medium buttons, so the intent (uniform, substantial buttons) survives and long labels still fit. Root admin.jsx and panels.jsx are deleted. The kit is the single source.
Verdict

No P1s remain. The states pass and the accessibility pass hold under re-measurement. The remediation plan's exit condition is met.
